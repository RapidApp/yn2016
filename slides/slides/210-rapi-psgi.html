rapi/psgi

  • Public Docker Hub Image
  • Designed to run any PSGI application (not just RapidApp)
  • Lots of functionality
    • Use in your Dockerfile, or directly
    • Automatically runs app.psgi (i.e. plackup)
    • Uses high performance prefork HTTP server (Gazelle)
    • Optionally automatically installs missing deps (cpanfile)
    • Many configuration options
    • Well-documented
