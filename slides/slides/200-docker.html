RapidApp + Docker

  • Traditional installation:
    • cpanm RapidApp
    • Installs the whole (missing) dep tree from CPAN on your system
    • Can take a long time (1+ hours)
    • Might fail (upstream distributions are broken from time to time)
  • Docker installation:
    • docker pull rapi/psgi
    • Pre-installed image of full perl/RapidApp stack
    • Takes seconds/minutes to download
    • Known tested/working state
    • Isolated environment
